TRELLO
trello é um ferramenta de gerenciamento de projetos baseada em quadros, que permite aos usuários criar, organizar e priorizar tarefas e informações em um formato visual e intuitivo
CANVAS
Canvas é uma ferramenta de planejamento estratégico que ajuda os empreendedores a visualizar e desenvolver um modelo de negócios de maneira simples e organizada.
MY SQL
MySQL é um sistema de gerenciamento de banco de dados relacional que fornece uma plataforma para armazenar, organizar e acessar dados de maneira eficiente
VISUAL STUDIO
O Visual Studio Code é um editor de código-fonte gratuito e de código aberto que oferece uma plataforma para desenvolvedores criarem, depurarem e testarem aplicativos de software..
FIGMA
O Figma é uma plataforma de design de interface do usuário (UI) e user experience (UX) baseada em nuvem, que permite aos designers criar, compartilhar e colaborar em projetos de design de maneira eficiente
DISCORD
O Discord é um aplicativo de comunicação baseado em nuvem que permite aos usuários se comunicarem por meio de mensagens de texto, voz e vídeo
